# 프로젝트 요약 (제출용)

## 프로젝트명
**계량적 요인 변화에 따른 이직/퇴사 리스크 예측 대시보드**

## 1) 문제정의
HR 운영에서 이직 리스크를 정량적으로 조기 탐지하고,
특히 **보상(절대/상대 임금) 레버리지 관점**에서 우선 대응 대상을 식별하기 위한 분석 도구를 구축했습니다.

## 2) 내가 해결한 핵심 질문
- 어떤 구성원이 향후 1년 내 이직 위험이 높은가?
- 부서/직급 단위에서 리스크가 집중되는 영역은 어디인가?
- 보상 조정은 어디(절대 고/상대 저 구간)에 우선 투입해야 효과적인가?

## 3) 접근 방법
- 모델: 로지스틱 회귀 (해석가능성 중심)
- 검증: Hold-out + 반복 split + Stratified K-fold
- 해석: 오즈비(OR), SHAP, 단건 기여도
- 시각화: 개인/조직 리스크, 임금 경쟁력, 4분면 분석

## 4) 주요 구현 포인트
- ERP 데이터 컬럼 자동 매핑(한글 컬럼명 대응)
- 산업 평균 임금 데이터 결합(year, tenure 기준 최근접 매칭)
- 절대임금(salary_std)과 상대임금(log_ratio) 동시 반영
- salary_std와 산업상대임금 간 공선성 완화를 위한 잔차화(orthogonalization)
- 재직자 기준 부서/직급 리스크 자동 집계
- What-if 시뮬레이션으로 정책 시나리오 즉시 테스트

## 5) 산출 화면
1. 개인 단위 예측(퇴사확률/위험도)
2. 조직별 리스크(부서/직급 평균확률, High/Medium/Low 분포)
3. 임금 경쟁력(산업 대비/내부 대비 배수)
4. 절대임금×상대임금×이직확률 4분면
5. 모델 심층탭(OR/SHAP/검증)

## 6) 기대 활용
- 리텐션 대상 우선순위 설정
- 보상·승급·직무전환 정책 타깃팅
- 월/분기 단위 HR 리스크 모니터링

## 7) 역할 및 제작 방식
- 문제정의, 변수설계, 해석프레임, 정책 연결은 직접 설계
- 코드 작성은 AI-assisted 방식 일부 활용
- 결과 해석과 의사결정 제안은 직접 검토/책임

## 8) 한계 및 보완 계획
- 단일 모델 기반 → 향후 트리계열/서바이벌 분석 병행 예정
- 표본 편향 가능성 → 기간 확장 및 외부 검증 필요
- 인사 의사결정 단독근거 사용 금지, 정성평가 병행 원칙 유지
