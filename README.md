# 이직/퇴사 리스크 예측 대시보드 (Streamlit)

ERP 인사 데이터와 산업 평균 임금 데이터를 결합해, 구성원/조직 단위의 이직(퇴사) 리스크를 예측하고 보상 레버리지 포인트를 시각화하는 대시보드입니다.

## 1) 주요 기능

- 개인 단위 **퇴사 확률 예측** (What-if 시뮬레이션)
- 부서/직급별 **High·Medium·Low 리스크 분포**
- 산업 대비/내부 대비 임금 경쟁력 분석
- 절대연봉 × 상대연봉 × 이직확률 **4분면 분석**
- 모델 해석: 오즈비(OR), SHAP, 반복검증/교차검증

## 2) 실행 방법

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# Mac/Linux
# source .venv/bin/activate

pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 기본적으로 `http://localhost:8501` 로 열립니다.

## 3) 입력 데이터 형식

### (A) ERP 데이터 (xlsx/csv)
필수 컬럼(한글 컬럼명 자동 인식):

- 성과등급
- 근속기간
- 성별
- 연령
- 통근시간
- 연봉(만원)
- 직무만족도
- 기준연도(pay_year)
- 퇴사여부(attrition, 0/1)

선택 컬럼:

- 사번(또는 직원번호)
- 부서(또는 소속부서, 팀)
- 직급(또는 직위)

### (B) 산업 평균 임금 CSV
헤더 예시:

```csv
year,tenure,industry_avg_wage
2024,1-3y,4200
2024,3-5y,5100
```

> `year`, `tenure`, `industry_avg_wage` 3개 컬럼이 필요합니다.

## 4) 산출 화면 요약

- **단건 예측:** 현재 입력 조건에서 향후 1년 내 퇴사 확률 추정
- **조직 리스크:** 부서/직급별 평균 예측 확률 + 위험도 구간 인원
- **임금 경쟁력:** 산업 대비 배수, 직급 내 배수 비교
- **4분면 분석:** 절대연봉(연도 내 z-score)과 상대임금 기준 보상 우선 타깃 탐색
- **심층 분석:** 오즈비(계수표), SHAP 중요도, 검증지표(AUC/Brier)

## 5) 해석 시 유의사항

- 예측확률은 **통계적 리스크 지표**이며 확정 예언이 아닙니다.
- 민감정보(사번, 실명, 원본 연봉)는 외부 공유 시 반드시 마스킹하세요.
- 정책 의사결정은 조직/직무 맥락 및 현업 인터뷰와 함께 해석해야 합니다.

## 8) Known Limitations

- 단일 모델(로지스틱 회귀) 중심이라 비선형 관계를 충분히 반영하지 못할 수 있음

- 정책 적용 전, 현업 인터뷰/정성평가/법적·윤리적 검토가 필요
